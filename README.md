## Dev setup

 * Install mongodb - [download](http://www.mongodb.org/downloads)

 * Install dependencies

    ```
    $ pip install -r requirements.txt
    ```

 * Load data

    ```
    $ PYTHONPATH=. python data/load.py
    ```

 *  Go to [localhost](http://localhost:5000)

    ```
    $ python runserver.py
    ```
